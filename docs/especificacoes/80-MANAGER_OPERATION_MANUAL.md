# 80-MANAGER_OPERATION_MANUAL.md

# Manual Operacional do Gestor

## Objetivo

Orientar o gestor na utilização do CarCenter PRO Finance como ferramenta
de tomada de decisão, acompanhamento financeiro e gestão da operação.

------------------------------------------------------------------------

# Perfil do Gestor

Responsável por:

-   Acompanhar indicadores.
-   Validar resultados financeiros.
-   Monitorar despesas.
-   Controlar orçamento.
-   Encerrar competências.
-   Apoiar decisões estratégicas.

------------------------------------------------------------------------

# Rotina Diária

Ao iniciar o expediente:

1.  Acessar a HOME.
2.  Verificar alertas do Assistente Gerencial.
3.  Conferir KPIs:
    -   Receita
    -   Despesas
    -   Resultado
    -   Projeção
4.  Avaliar pendências financeiras.

------------------------------------------------------------------------

# Acompanhamento do Dashboard

Verificar diariamente:

-   Receita acumulada.
-   Despesa acumulada.
-   Resultado operacional.
-   Ticket médio.
-   Tendência mensal.
-   Centros de custo.

Utilizar filtros para comparar períodos.

------------------------------------------------------------------------

# Gestão das Receitas

Conferir:

-   Lançamentos do dia.
-   Categorias com maior faturamento.
-   Ticket médio.
-   Evolução da receita.

Quando identificar inconsistências:

-   Revisar lançamentos.
-   Solicitar correções quando necessário.

------------------------------------------------------------------------

# Gestão das Despesas

Monitorar:

-   Parcelas vencidas.
-   Pagamentos do dia.
-   Despesas por categoria.
-   Despesas extraordinárias.

Priorizar pagamentos críticos.

------------------------------------------------------------------------

# Controle Orçamentário

Semanalmente:

-   Comparar Orçado x Realizado.
-   Identificar categorias acima do orçamento.
-   Definir ações corretivas.

Quando houver excesso:

-   Avaliar causa.
-   Registrar plano de ação.

------------------------------------------------------------------------

# Análise da DRE

Ao final de cada competência:

Verificar:

-   Receita Bruta.
-   Receita Líquida.
-   Despesas Operacionais.
-   Resultado Operacional.
-   Margem.

Confirmar consistência antes do fechamento.

------------------------------------------------------------------------

# Fechamento da Competência

Checklist:

-   Todas as receitas registradas.
-   Pagamentos conciliados.
-   Pendências resolvidas.
-   DRE conferida.
-   Orçamentos revisados.

Após validação:

-   Encerrar competência.
-   Registrar decisão.

------------------------------------------------------------------------

# Relatórios

Gerar periodicamente:

-   DRE.
-   Receitas.
-   Despesas.
-   Orçamentos.
-   Fluxo Financeiro.

Exportar em PDF para apresentação gerencial quando necessário.

------------------------------------------------------------------------

# Assistente Gerencial

Priorizar alertas na ordem:

1.  Críticos.
2.  Altos.
3.  Médios.
4.  Baixos.

Resolver pendências antes do fechamento da competência.

------------------------------------------------------------------------

# Boas Práticas

-   Consultar indicadores diariamente.
-   Evitar decisões sem analisar tendências.
-   Revisar orçamento mensalmente.
-   Manter competências encerradas corretamente.
-   Utilizar relatórios como apoio às reuniões.

------------------------------------------------------------------------
