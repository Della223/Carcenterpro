# 81-FINANCIAL_OPERATION_MANUAL.md

# Manual Operacional do Setor Financeiro

## Objetivo

Orientar os usuários do setor financeiro na execução das rotinas
operacionais do CarCenter PRO Finance, garantindo padronização,
consistência dos dados e conformidade com as regras de negócio.

------------------------------------------------------------------------

# Perfil do Usuário

Responsável por:

-   Registrar receitas.
-   Registrar despesas.
-   Gerenciar parcelamentos.
-   Registrar pagamentos.
-   Conferir movimentações.
-   Apoiar o fechamento da competência.

------------------------------------------------------------------------

# Rotina Diária

1.  Acessar a HOME.
2.  Verificar notificações.
3.  Conferir parcelas vencidas.
4.  Conferir pagamentos previstos.
5.  Registrar movimentações pendentes.

------------------------------------------------------------------------

# Cadastro de Receitas

Fluxo:

1.  Abrir Comercial.
2.  Nova Receita.
3.  Informar data, categoria, quantidade e valor.
4.  Salvar.

Após salvar:

-   Atualizar HOME.
-   Atualizar Dashboard.
-   Atualizar DRE.
-   Registrar auditoria.

------------------------------------------------------------------------

# Cadastro de Despesas

Fluxo:

1.  Abrir Despesas.
2.  Nova Despesa.
3.  Informar fornecedor, categoria, centro de custo e valor.
4.  Definir parcelamento.
5.  Salvar.

O sistema gera automaticamente as parcelas.

------------------------------------------------------------------------

# Registro de Pagamentos

1.  Selecionar parcela.
2.  Registrar pagamento.
3.  Confirmar.

Após confirmação:

-   Atualizar DRE.
-   Atualizar Dashboard.
-   Registrar auditoria.

------------------------------------------------------------------------

# Conciliação

Conferir diariamente:

-   Receitas.
-   Despesas.
-   Pagamentos.
-   Pendências.

------------------------------------------------------------------------

# Fechamento

Antes do encerramento:

-   Validar receitas.
-   Validar despesas.
-   Revisar parcelas.
-   Conferir DRE.

------------------------------------------------------------------------

# Relatórios

Utilizar:

-   Receitas
-   Despesas
-   Fluxo Financeiro
-   DRE

------------------------------------------------------------------------

# Boas Práticas

-   Conferir dados antes de salvar.
-   Evitar registros duplicados.
-   Registrar pagamentos imediatamente.
-   Não compartilhar credenciais.

------------------------------------------------------------------------

# Checklist

## Diário

-   Receitas registradas.
-   Despesas registradas.
-   Pagamentos atualizados.

## Mensal

-   Conciliação concluída.
-   DRE conferida.
-   Competência pronta para encerramento.

------------------------------------------------------------------------

# Critérios de Aceite

-   Manual compatível com a versão 1.0.
-   Linguagem operacional.
-   Procedimentos alinhados às regras de negócio.
