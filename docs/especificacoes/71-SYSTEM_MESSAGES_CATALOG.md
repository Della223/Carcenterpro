# 71-SYSTEM_MESSAGES_CATALOG.md

# Catálogo Oficial de Mensagens do Sistema

## Objetivo

Padronizar todas as mensagens exibidas pelo CarCenter PRO Finance,
garantindo comunicação consistente, clara e em Português (Brasil).

------------------------------------------------------------------------

# Princípios

-   Linguagem simples e objetiva.
-   Evitar termos técnicos para o usuário final.
-   Informar o problema e, quando possível, como resolvê-lo.
-   Nunca expor detalhes internos do sistema.

------------------------------------------------------------------------

# Diretrizes de Escrita

-   Português do Brasil.
-   Voz ativa.
-   Frases curtas.
-   Não utilizar abreviações desnecessárias.
-   Padronizar termos em todos os módulos.

------------------------------------------------------------------------

# Critérios de Aceite

-   Todas as mensagens centralizadas.
-   Nenhum texto fixo espalhado pelo código.
-   Catálogo utilizado por toda a aplicação.
