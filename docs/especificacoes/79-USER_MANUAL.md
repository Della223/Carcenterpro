# Manual do Usuário

## Objetivo

Orientar os usuários do CarCenter PRO Finance na utilização diária da
plataforma em linguagem simples e operacional.

------------------------------------------------------------------------

# Primeiro Acesso

1.  Acesse a URL do sistema.
2.  Informe e-mail e senha.
3.  Clique em **Entrar**.
4.  Ao autenticar, a HOME será exibida.

Caso esqueça a senha, utilize **Esqueci minha senha**.

------------------------------------------------------------------------

# Navegação

Menu principal:

-   HOME
-   Comercial
-   Despesas
-   Dashboard
-   DRE
-   Orçamentos
-   Relatórios
-   Configurações

O menu permanece disponível durante toda a navegação.

------------------------------------------------------------------------

# HOME

Na HOME o usuário poderá:

-   Visualizar KPIs.
-   Consultar pendências.
-   Ver alertas do Assistente Gerencial.
-   Acessar rapidamente outros módulos.

------------------------------------------------------------------------

# Comercial

Permite:

-   Cadastrar receitas.
-   Editar receitas.
-   Excluir receitas (conforme permissão).
-   Filtrar informações.
-   Exportar dados.

Fluxo:

1.  Clique em **Nova Receita**.
2.  Preencha os campos obrigatórios.
3.  Clique em **Salvar**.

------------------------------------------------------------------------

# Despesas

Permite:

-   Cadastrar despesas.
-   Parcelar valores.
-   Registrar pagamentos.
-   Consultar histórico.

Fluxo:

1.  Clique em **Nova Despesa**.
2.  Informe fornecedor, categoria e valor.
3.  Defina o parcelamento.
4.  Salve o registro.

------------------------------------------------------------------------

# Dashboard

Permite acompanhar:

-   Receita.
-   Despesas.
-   Resultado.
-   Gráficos.
-   Indicadores.

Utilize os filtros para alterar o período analisado.

------------------------------------------------------------------------

# DRE

Exibe automaticamente:

-   Receita Bruta.
-   Receita Líquida.
-   Despesas.
-   Resultado Operacional.

Os valores são calculados automaticamente.

------------------------------------------------------------------------

# Orçamentos

Permite:

-   Criar orçamento anual.
-   Comparar realizado x planejado.
-   Acompanhar alertas.

------------------------------------------------------------------------

# Relatórios

Selecione:

-   Tipo.
-   Período.
-   Filtros.

Depois clique em **Exportar** ou **Imprimir**.

------------------------------------------------------------------------

# Configurações

Disponível apenas para usuários autorizados.

Permite administrar:

-   Categorias.
-   Subcategorias.
-   Centros de Custo.
-   Competências.
-   Parâmetros.

------------------------------------------------------------------------

# Boas Práticas

-   Sempre confirme os dados antes de salvar.
-   Não compartilhe sua senha.
-   Utilize os filtros para localizar informações.
-   Faça logout ao finalizar o uso.

------------------------------------------------------------------------

# Perguntas Frequentes

## Não consigo acessar.

Verifique usuário e senha ou utilize a recuperação de acesso.

## Não encontro um lançamento.

Revise os filtros ativos e a competência selecionada.

## Posso editar a DRE?

Não. A DRE é calculada automaticamente.

------------------------------------------------------------------------

# Suporte

Em caso de dúvidas ou falhas, entre em contato com o administrador do
sistema.
